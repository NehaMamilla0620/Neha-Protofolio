# [Neha Mamilla] | Personal Portfolio

A clean, responsive portfolio website built from scratch to showcase my projects, skills, and experience.

## 🚀 Live Demo
[View Live Site]https://nehamamilla0620.github.io/Neha-Protofolio/)

## 🛠️ Tech Stack
* **HTML5**: Semantic structure for better SEO.
* **CSS3**: Custom styling and layouts using Flexbox/Grid.
* **JavaScript**: Interactive elements and smooth scrolling.
* **Deployment**: GitHub Pages.

## ✨ Features
* **Fully Responsive**: Optimized for mobile, tablet, and desktop screens.
* **Project Showcase**: A curated list of my best development work.
* **Clean UI**: Minimalist design focused on readability and user experience.
* **Contact Section**: Ways to reach out to me via email or linkedin links.

## 🏗️ How to Run Locally
1. Clone the repository:
   ```bash
   git clone https://github.com
   ```
2. Navigate to the project folder:
   ```bash
   cd repository-name
   ```
3. Open `index.html` in your favorite browser.
